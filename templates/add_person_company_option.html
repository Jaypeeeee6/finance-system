Person/Company Name Guidelines

Naming Conventions:
  • Use the exact company or person name as it should appear
  • Include any location or branch information if needed
  • Keep names clear and consistent
  • Use proper capitalization
Department & Request Type:
  • Each option is specific to a department and request type combination
  • Different departments can have different options for the same request type
  • Users will only see options for their department and selected request type
  • Choose the most appropriate department and request type
Tip: You can always edit or deactivate person/company name options later if needed. Deactivated options will not appear in the dropdown but will remain in the database for historical records.
