Request Type Guidelines

Naming Conventions:
  • Use clear, descriptive names
  • Keep names concise but informative
  • Use appropriate capitalization (e.g., "Personal Expenses", "AC Installation")
  • Avoid abbreviations when possible
Department Assignment:
  • Each request type belongs to one department
  • Users from that department will see this type
  • General Manager role can see all types
  • Choose the most appropriate department
Tip: You can always edit or deactivate request types later if needed. However, you cannot delete request types that are being used by existing payment requests.
